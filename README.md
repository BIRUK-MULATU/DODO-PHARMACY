# RxPharma — Pharmacy Management System

A full-stack pharmacy management system built with Spring Boot and React.

## Tech Stack

### Backend
- Java 21 + Spring Boot 3.4.5
- PostgreSQL + Flyway migrations
- JWT Authentication + Spring Security
- Docker Compose

### Frontend
- React + Vite
- Tailwind CSS
- Axios

## Project Structure
rxpharma/
├── rxpharma-backend/
└── rxpharma-frontend/

## Getting Started

### Backend
cd rxpharma-backend
docker compose up -d
./mvnw spring-boot:run

### Frontend
cd rxpharma-frontend
npm install
npm run dev
